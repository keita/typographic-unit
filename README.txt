=README

TypographicUnit is a library for converting between typographic units according
to TeX.

== Units

* TeX Scaled Point(sp)
* TeX Point(pt)
* Pica(pt)
* Inch(in)
* TeX Big Point(bp)
* PostScript Point(ps_pt)
* Meter(m)
* Centimeter(cm)
* Milimeter(mm)
* Didot Point(dd)
* Cicero(cc)
* Japanese Q(q)
* JIS Point(jis_q)
